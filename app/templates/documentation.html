Introduction

SportTracker Pro est une plateforme complète de gestion de performances sportives. Elle offre des outils avancés pour le suivi GPS, la gestion médicale, et l'analyse des données d'entraînement.

Installation

Suivez ces étapes pour installer SportTracker Pro:

# Cloner le projet
git clone https://github.com/your-org/sporttrackerpro.git
cd sporttrackerpro

# Créer l'environnement virtuel
python -m venv .venv
source .venv/bin/activate

# Installer les dépendances
pip install -r requirements.txt

# Lancer le serveur
python run.py

Rôles & Permissions

SportTracker Pro supporte 8 rôles différents:

  • Admin: Accès complet à toutes les fonctionnalités
  • Coach: Gestion des équipes et des séances
  • Préparateur: Suivi de la préparation physique
  • Médecin: Gestion des blessures et du wellness
  • Analyste: Analyse des données et rapports
  • Joueur: Accès à son profil personnel
  • Assistant: Support aux coachs
  • Dirigeant: Vue managériale des équipes

API REST

SportTracker Pro expose une API REST complète:

GET /api/players - Liste tous les joueurs
GET /api/players/{id} - Détails d'un joueur
POST /api/players - Créer un joueur
PUT /api/players/{id} - Mettre à jour
DELETE /api/players/{id} - Supprimer

GET /api/teams - Liste les équipes
GET /api/sessions - Liste les séances
GET /api/gps-data - Données GPS

Questions Fréquentes

Quels formats de données GPS sont supportés?

Nous supportons les fichiers CSV, JSON et les uploads directs d'appareils GPS.

Comment seguriser mes données?

Toutes les données sont chiffrées et stockées sur des serveurs sécurisés.

Ya-t-il une limite de joueurs?

Non, la plateforme scale automatiquement selon vos besoins.
